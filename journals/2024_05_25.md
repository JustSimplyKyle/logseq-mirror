- # Rust 2 Cpp
	- ## 輸入輸出
		- ```cpp
		  int main() {
		  	int a; // declaration without initialization
		    	int b;
		      cin >> a >> b; // right arrow
		  	cout << a;
		  }
		  ```
		- cin 在新行或空格才會放入變數
	- ## if 判斷式
		-
		- ```cpp
		  int main() {
		  	int a = 5;
		    	int b = 5;
		    	if (a==5 && b == 5) {
		        	cout << "success"
		      }
		  }
		  ```
		- if 判斷式需加括號
