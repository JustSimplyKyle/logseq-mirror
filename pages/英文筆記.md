	- ## When to use -sion
	  collapsed:: true
		- words ending with -de
			- decide -> decision
			- explode -> explosion
			- persuade -> persuasion
			- invade -> invasion
	- ## "Important" Synonyms
	  collapsed:: true
		- essential  必要的
		- crucial 必須的
		- vital 必要的
		- fatal 嚴重的（負面）
		- significant 顯著的
	- ## the abroad family
	  collapsed:: true
		- abroad -> 國外
		- aboard -> 船上
		- board -> 板子
		- broad -> 寬廣
