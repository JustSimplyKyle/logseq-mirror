# 安裝 `linux`/`wsl`
	- 打開 `powershell`，填入 `wsl --install`
- # 你需要知道的指令
	- ## 基礎指令(操縱你所在的資料夾的資料)
		- ### `ls`
			- ``--color``
			- `-l`
			- `../`
			- `./cp` vs `cp`
		- ### `pwd`
		- ### ``cd [filepath]``
		- ### ``cat [filepath]``
		- ### `mkdir [filepath]`
		- ### `rm [filepath]`
			- `-rf`
		- ### `touch [filename]`
		- ### ``nano [filename]``
	- ### `chmod`
		- `-R`
		- `600`
		- `755`
		- `777`
		- `+x`
	- ### `wget`
- # 腳本撰寫
	- 跟寫 python 很像，每一行順序執行
	- #!/bin/bash
	  
	  ls
	  mkdir my_path
	  cd my_path
	  pwd
	- 利用 `myVariable=1234` 來創建變數
	- 利用 ``
	  ```bash 
	  myVariable=`whoami`
	  ```
	  來紀錄指令的輸出

		- 利用 `./redirect` 得到輸出，並將其輸出「redirect」 到 `redirect.txt`
		- 再執行 `./redirect` 得到 flag
