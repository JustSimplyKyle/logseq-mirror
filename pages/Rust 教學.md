# 宣告變數
	- ## 使用方法
		- 在 Rust 中，所有變數都有 **型別** ，但是你可以不用特別把型別寫出來，編譯器會**自動**幫你推斷。
			- ```rust
			  let x = 42;	
			  ```
		- 也可以將型別手動標記出來。
			- ```rust
			  // i32 代表 32 位整數
			  let x: i32 = 42;
			  ```
		- 在 Rust，在預設情況話，所有變數都是 **不可變** 的，需利用 `let mut` 來宣告
			- 但理想情況下，我們要盡量**減少** `mut` 的使用，減少所需要在腦中管理的 **state**
			- ```rust
			  let x = 32;
			  x = 87; // 錯誤
			  let mut y = 32;
			  y = 72; // ok
			  ```
		- Rust 同時也支援 **Variables Shadowing**
			- ```rust
			  let x = 32;
			  let x = 64;
			  // x == 64
			  ```
